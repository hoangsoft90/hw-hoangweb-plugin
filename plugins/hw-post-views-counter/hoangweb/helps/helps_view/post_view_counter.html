Shortcode

Use [post-views] shortcode for manual display. [hoangweb] - to display post views count, using following shortcode: [hw-post-views]: for current post [hw-post-views id=12] : for given post [hw-post-views id=13 title="post title" permalink="any link"] : full shortcode attributes

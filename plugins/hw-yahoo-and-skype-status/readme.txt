Display yahoo skype status with flexible templates

== Description ==
Show list yahoo skype nick status using Widget.

== Installation ==
1. "HW SKIN" is requiring plugin in order to active this plugin.
Firstly, you need to download & active HW SKIN plugin.

2. Active "HW Yahoo Skype Status Pro" plugin. After plugin installed, a new widget named "Online yahoo skype status" added to widgets page, use the widget to add on your sidebar to display chat nicks with yahoo, skype service.

== Usage ==
Note: when first adding widget you need to save before able to setup widget.
1. Go to appearance --> Widgets and drang HW Yahoo Skype status Pro to your sidebar. Note: you save widget for first
2. Enter your name, yahoo/skype ID, Your fullname and other options
3. Save

* Adding Images:
You can upload your own icons for phone, mail, yahoo/skype status image alternate. By copy & paste your images into /skins/resources folder.

* Create new template:
Create your skin in /skins folder, each skin present a folder. Extend default skin in this plugin to learn more about skin creation.
Extend skin path: Plugin will load directory of skin in different folder, may be check & loaded in bellow locations:
- skins/ (refer to current plugin.)
- wp-content/hw_yahooskype
- themes/your-theme/hw_yahooskype

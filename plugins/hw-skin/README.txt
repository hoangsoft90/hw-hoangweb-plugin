== Description ==
A PHP class allow you to create your widget content template

== Installation ==
1. Upload the entire hw-skin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

read document.chm file to learn about how to create template for your widget content.
